Biominers/covid19-testing-data: internal dashboard

This is part of Khazen Lab's SARS-CoV-2 Testing Data.

  • p1a: Sources over time: Stacked

    This is the plot in our pub #1. It shows the number of points coming from each source per country/state/date.

  • p1b: Sources over time: Line

    This complements the stacked plot above, and was not used

  • p2: Postprocessing

    This shows the details of the postprocessing: interpolation followed by easing spikes. For example, it shows clearly that most of our data is artificial for Kuwait as of 2020-05-12. For a list of countries with spike easing, check the data repo file "l2/interpolation_peakstats.csv". Example countries with special spike consideration: Austria (1 peak), Liechenstein (no peaks), Zimbabwe (should have no peaks), France (starting point could yield cases being larger than tests), China - Hubei (starting point of data is not 0).

  • Chi-squared min/max thresholds for testing

    • p3b: detailed

      Calculates min/max number of cases for each corresponding number of tests under a p-value of 5% such that the 7-day moving sum is in-line with the 14-day moving sum. If the number of cases is below the minimum, then the country is doing well. If it is above it, then the country is not doing well.

    • p3a+p4: with slopes scatter plot

      Same as the detailed plot above, but kept only 2 plots of all, and added the slopes scatter plot by Halim showing the change in cases versus the change in testing for each country.

    • Animation of past 7-day improvement vs current 7-day excess cases (from notebook shadi/t17b): per continent, per country

      These are animations of the scatter plots in the detail page above. The scatter plot is just the last entry for the detail page, but in this animation it shows the full history.

  • p5: Global scatter plots of tests vs cases and diff tests vs diff cases

    Helps find countries that are good and improving or bad and improving or ...

  • Association Analysis between genomics and epidemic metrics

    Correlation of epidemic metrics with genomics. This particular visualization was done by Rim and Rasha on 2020-05-20. Note that this is not generated by this repo, but by the covid19 repo in Rim's gitlab repo ATM

  • html table of latest numbers per country/state

    By Halim, 2020-05-28. Serves as a sortable table of latest results.
